# 🤖 AI Email Generator

An AI-powered email generator built using **React**, **Node.js**, **Express**, and **Google Gemini AI**. This application generates professional emails based on the recipient, email type, tone, and purpose provided by the user.

---

## ✨ Features

- 🤖 AI-generated professional emails
- 📧 Multiple email types
- 🎯 Tone selection (Professional, Friendly, Formal)
- 👤 Recipient name input
- 📝 Purpose-based email generation
- 📋 Copy generated email
- 🗑️ Clear form
- ⏳ Loading indicator while generating emails
- 🎨 Clean and responsive user interface

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)

### Backend
- Node.js
- Express.js

## 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/Anusha-0811/ai-email-generator.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
node index.js
```
